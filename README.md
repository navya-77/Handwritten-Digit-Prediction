# Handwritten Digit Prediction

## Overview
This project is a Deep Learning-based handwritten digit recognition system developed using Convolutional Neural Networks (CNN). The model predicts handwritten digits from image inputs with high accuracy.

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Matplotlib

## Features
- Handwritten digit image prediction
- CNN-based classification
- Image preprocessing
- High accuracy digit recognition
- Deep learning model training

## Dataset
MNIST Handwritten Digit Dataset

## Model
Convolutional Neural Network (CNN)
